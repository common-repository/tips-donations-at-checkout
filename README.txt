== Description ==

The plugin simply adds "Tips" field to your website's WooCommerce Checkout Form. 
It can add it as a native fee and via the Notice box as well.

Once the plugin is installed and activated, you will have a new field in the Checkout Form where your customers can add optional tip&donation.

== Installation ==

1. Upload `tips-donations-at-checkout.zip` to the `/wp-content/plugins/` directory or Use Wp-Admin->Plugins->Add New and upload tips-donations-at-checkout.zip
2. Activate the plugin using the 'Plugins' menu in your WordPress website.
3. Navigate to WP-Admin->WooCommerce->Tips&Donations to get started.

== Frequently Asked Questions ==

= Is the added donation field required? =

No, it isn't. Customers can set it to 0(zero) and not to pay any tip. And that Zero can be the default value as well.

= Is the plugin freemium? =

Currently it is absolutely free and doesn't have a PRO version.
